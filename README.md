# Conversor
Conversor de monedas Alura Challenge
Este proyecto consta de un conversor de varios tipos de moneda a pesos mexicanos y viceversa, 
como extra se añadio un conversor de pesos de gramos a libras y onzas y viceversa.

#### Recursos:
- Lenguaje: Java.
- IDE: IntelliJ.
